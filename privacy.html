Mailtropy Privacy Policy

Effective Date: [2026-04-12]

1. Overview

Mailtropy is a Chrome extension that provides inbox composition analytics for Gmail users. All analysis is performed locally within the user's browser. Mailtropy stores analysis metadata only in local extension storage and does not transmit or sell email data.

2. Data Access

Mailtropy requests read-only access to Gmail using the following scope:

https://www.googleapis.com/auth/gmail.readonly

Mailtropy accesses only:

  • Message ID
  • Thread ID
  • Timestamp
  • Label IDs
  • From header
  • List-Unsubscribe header

Mailtropy does not access:

  • Email body content
  • Attachments
  • Email snippets
  • Email subject lines

3. Data Storage

All analysis occurs locally within the Chrome extension. Mailtropy does not transmit email data to external servers and does not use remote databases.

4. Data Sharing

Mailtropy does not sell, share, or transfer user data to third parties.

5. User Control

Mailtropy does not modify, delete, archive, or unsubscribe users from emails. All cleanup actions are performed manually by the user within Gmail.

6. Data Retention

Email metadata remains in local extension storage. Users may remove all stored data by uninstalling the extension.

7. Security

Mailtropy follows a zero-server architecture. No backend infrastructure exists.
